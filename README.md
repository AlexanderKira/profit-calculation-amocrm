# Установка проекта
Копирование репозитория

    git clone https://github.com/AlexanderKira/profit-calculation-amocrm.git
Зайти в проект

    cd message-channel
Скопировать env.example в .env

    cp .env.example .env
Установить composer

    composer install
Зайти в env и установить значения
Конфигурация бд

    DB_CONNECTION=mysql
    DB_HOST=mysql
    DB_PORT=3306
    DB_DATABASE=laravel
    DB_USERNAME=sail
    DB_PASSWORD=password

Зайти в контейнер

    docker-compose up -d
    
    php artisan migrate

    npm run dev
